# Age of Crowns

A medieval turn-based strategy game of world conquest. Play in your browser, no install needed.

Lead one of 8 historical empires across a real world map of 40 provinces. Raise armies of spearmen, swordsmen, longbows, crossbows, cavalry, and catapults. Hold 60% of the world to win.

**Play:** open `index.html`, or visit the GitHub Pages URL for this repository.
